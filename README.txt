Additional features:

1. Toggle 'L': enable lightspeed travel.
2. Toggle 'F': enable weapon system on Enterprise. 
3. Toggle 'E': enable cloaking on the Enterprise.
3. Flashing effect on stars randomly, this can be seen by toggling the 'S' key.
4. Saturn rings, this can be seen after the lightspeed travel.
